About This Project

This Wikipedia Clone was created by Kashaf Fatima as part of the SEL-310 Web Engineering course. The goal was to replicate a real-world website using only HTML and CSS, while practicing responsive design, semantic markup, and clean coding principles.

Key Features

  • Responsive layout using Flexbox & Grid
  • Semantic HTML5 structure
  • Mobile-friendly navigation bar
  • Wikipedia-powered search functionality
