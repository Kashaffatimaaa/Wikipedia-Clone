Help & Support

This page provides guidance on how to use the Wikipedia Clone. Since this is a student project, features are simplified but still modeled after the real Wikipedia.

How to Search

Use the search bar at the top to find Wikipedia articles. If the article exists, it opens directly; otherwise, you’ll be redirected to Wikipedia’s search results.
